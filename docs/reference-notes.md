# Reference Notes

Reviewed June 10, 2026.

## The Daily Hue

High-level principles observed from the public game and learning pages:

- Make the featured image the primary visual object.
- Explain HSL in familiar language before introducing precision.
- Keep the five-attempt loop visible and predictable.
- Put directional clues beside the controls they describe.
- Reveal cultural/source context after the game state resolves.
- Make zoom, attribution, spoiler-free sharing, and streak information easy to
  find without crowding the active guess.

AstroHue deliberately differs in identity, wording, page composition,
typography, colors, iconography, motion, and result layout. No reference assets
or source code are included.

## NASA Guidance

NASA states that its content is generally available for educational or
informational use with acknowledgement and without implied endorsement, while
also warning that some hosted material belongs to third parties. NASA
identifiers and logos are protected. Therefore every record remains subject to
manual rights and attribution review, and AstroHue includes a visible
independent-project disclaimer.
