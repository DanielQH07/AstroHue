# AstroHue MVP Plan

## Milestones

- [x] Project foundation, Next.js 16.2.9, strict TypeScript, Tailwind, design tokens
- [x] Private puzzle types and server-only generated data
- [x] NASA download, image processing, color extraction, and validation scripts
- [x] Color conversions, circular hue math, feedback, scoring, and selection
- [x] Stateless HMAC round tokens and validated Route Handlers
- [x] Responsive game interface and full-color image presentation
- [x] Result reveal, onboarding, zoom, local stats, persistence, and sharing
- [x] Accessibility semantics, keyboard controls, reduced motion, and mobile layout
- [x] Unit, component, storage, token, and API tests
- [x] Playwright browser matrix, visual fixes, production build, and final audit

Milestones are checked only after their corresponding automated verification
succeeds.
